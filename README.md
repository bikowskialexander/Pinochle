# Pinochle

## Description
Pinochle is a four person game with two teams. It is played with a different deck that contains nines, tens, aces, queens, jacks, and kings. There are still four suits and two of every card. 

This repository contains an user interface and game manager class to allow LLMs to play the game against a random bot, the user, or each other.

## LLM Control options
There are multiple methods for controlling the Pinochle game either with user input or llm generated responses.
